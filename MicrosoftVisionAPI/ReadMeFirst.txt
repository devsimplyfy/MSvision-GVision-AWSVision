Step 1: 
First download the project from Github.(repository: name)

Step 2: 
Open MicrosoftVisionAPI Folder and Find spring-cloud-gcp-vision-api-sample project from that bundle and import that project in spring-tool

Step 3: 
After importing you have set credential details in environment variables in PC setting and also in debug configuration.

Step 4: 
And then you have to clean and build maven.

Step 5 : 
After updating run that project as spring-boot application.

Step 6: 
Go to browser open localhost with specified port number. (port number is specified in       application.properties file). Browser give response with HTML Fpage.

Step 7: specify path and hit the submit button(According to controller).
